# ⚔️ The Spartan Rage

A brutal 2D action-adventure game inspired by the legendary **God of War–style** combat experience.  
**The Spartan Rage** delivers fast-paced hack-and-slash combat, cinematic boss fights, and a mythological revenge-driven storyline — all in a side-scrolling 2D world.

---

## 🎮 About The Game

**The Spartan Rage** is a 2D action platformer that blends:

- Fluid combo-based combat  
- Rage-powered special abilities  
- Massive boss encounters  
- Dark mythological storytelling  
- Cinematic pixel-art environments  

You play as a fallen Spartan warrior consumed by vengeance, battling gods, monsters, and titans in a relentless quest for redemption.

---

## 🕹️ Gameplay Features

- ⚔️ Combo-based melee combat system  
- 🔥 Rage Mode (temporary power amplification)  
- 🧌 Unique enemy types with custom AI  
- 🏛️ Mythology-inspired environments  
- 💀 Multi-phase boss fights  
- 🎵 Cinematic soundtrack  
- 💾 Save & progression system
